# basm - The assembler for BELLE


## Quickstart

`cargo build --release`

### CLI

| Field | CLI | Variable Type | Default Value | Example |
| :---  | :-- | :------------ | :------------ | :------ |
| Source code | `<file>` | String | `<none>` | `main.asm` |
| Binary output | `-o <file>` | String | `a.out` | `main.bin` |
| Verbose output | `-v` or `--verbose` | Boolean | `false` | `-v` |
